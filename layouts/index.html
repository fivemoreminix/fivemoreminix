I started programming as a 12 year old kid, tinkering with Visual Basic on my mom's old notebook. In seven years, I have feel that I have learned a lot about software development, without having taken any formal classes. I started my friends on programming, and now some of them do it as a hobby as well! Teach someone programming and positively affect their future.

This site uses no tracking, no cookies, and no JavaScript! I wrote the site from scratch using Hugo. You can find the site code on GitHub.
