I started programming as a kid, tinkering with Visual Basic. Today, I work with low level code to write high performance and experimental software for consumer computer systems. I'm passionately curious about computers and FOSS.

Medium generously pays me about $15/month for the articles I wrote on their site. While the passive income is nice, I don't believe it is approprite for my desire to teach and blog unbiased and without clickbait titles. This blog (lukewilson.me) is where I will be writing all new content. My old blog at Medium can be found here.

This site uses no tracking, cookies, nor JavaScript. You can see the site code on GitHub.
